Maid

Local AI Chat for Android

A free, open-source Android app that lets you chat with AI models entirely on your device — no subscription, no cloud, no data leaving your phone.

Features

Local & Remote Models

Run GGUF models on-device via llama.cpp, or connect to Ollama, OpenAI, Anthropic, Mistral, and DeepSeek with your own API key.

One-Tap Downloads

Browse and download curated Hugging Face models directly within the app — no manual file management needed.

Custom Model Support

Load any GGUF file from local storage. Full control over which model you run.

Configurable Parameters

Adjust temperature, top-p, top-k, context length, and other generation settings per session.

Chat Management

Create, rename, delete, export, and import conversations as JSON files.

Voice Integration

Pairs with Maise for on-device text-to-speech. Hear your AI respond without sending audio to the cloud.
